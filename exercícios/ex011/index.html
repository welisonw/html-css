Lista Ordenada

Para criar uma lista com ordem de numeração, basta utilizar a tag <ol>, que signigica ordered list:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Ligar para João

Ainda, é possível alterar o tipo do marcador. Para isso, basta adicionar o parâmetro type dentro da tag <ol>, atribuindo, como, por exemplo, a letra "A", para a sequência A, B, C..., ou, "I", para a sequência I, II, III...

Ainda, pode indicar o início da contagem usando o parâmetro start. Por exemplo, a tag <ol type=“I” start = “5”> vai gerar itens numerados como V, VI, VII, VIII, IX...

Lista não Ordenada

Para criar uma lista sem qualquer numeração ou ordem, basta utilizar a tag <ul>, de unordered list:

  • Ir ao mercado e comprar:
  • Pão
  • Leite
  • Queijo
  • Manteiga

O marcador padrão é a bolinha preta totalmente preenchida, mas existe a opção de configurar a propriedade type da tag <ul> com os seguintes valores:

    disc - padrão. Uma bola preta totalmente pintada.

    circle - Uma bola com uma borda preta e sem preenchimento.

    square - Um pequeno quadrado preto totalmente pintado.

Lista de definição

É como se fosse um dicionário, onde temos os termos e as suas descrições. É uma lista sem demarcadores, mas bem útil em alguns casos.

A lista de definições está dentro de uma tag <dl> (definition list). Cada termo é representado pela tag <dt> (definition term) e cada descrição pela tag <dd> (definition description):

HTML
Linguagem de marcação para a criação do conteúdo de um site
CSS
Linguagem de marcação para a criação do design de um site
JavaScript
Linguagem de programação para a criação da interatividade de um site
