Outras formatações

Código-fonte/Pré-formatação

O comando document.getElementById ('teste') é escrito em linguagem JavaScript. Essa formatação deixa a fonte mono-espaçada (ótima para ler códigos), indicando ao navegador que se trata de um código de computador. Basta utilizar a tag <code>.

Outro exemplo, citando trecho de um código da linguagem Python:

                    
                        num = int(imput ('Digite um número:'))
                        if num % 2 == 0
                            print(f'O número {num} é PAR!')
                        else:
                            print(f'O número {num} é ÍMPAR!')
                        print('Fim do programa')

Neste exemplo, foi utilizado a tag <code>. Contudo, na visualização, o código aparecerá desconfigurado, com as linhas uma ao lado da outra.

Para resolver isso, envelopar a tag <code> dentro da tag <pre>, que serve para mantém o texto pré-formatado, exatamente da mesma maneira na qual ele foi digitado, incluindo quebras de linhas, espaços e tabulações.

Citação simples

É possível, ainda, fazer citações utilizando a tag <q> (do inglês quote, que significa citar), como por exemplo, na seguinte frase:

O pai do meu amigo sempre dizia: O computador é um burro rápido.

Nesse exemplo, poderia simplesmente utilizar aspas (") no início e fim da frase. Porém, ao usar a tag Q, você atribui um significado à ela.

Citação composta/com recúo

Citações compostas utilizam a tag <blockquote>:

Seguro Aury Lopes Júnior, em seu livro Prisões cautelares:

A presunção de inocência impõe um verdadeiro dever de tratamento (na medida em que exige que o réu seja tratado como inocente), que atua em duas dimensões: interna ao processo e externa a ele.

Abreviação

Utilizar a tag <abbr>. Ao passar o mouse por cima da sigla, aparecerá um pop-up com seu significado:

Estou estudando HTML e CSS. Estou adorando!

Texto invertido

Para inverter o texto escrito, utilizar a tag <bdo>. BDO significa bi-directional override:

Estou aprendendo a criar coisas em HTML.
