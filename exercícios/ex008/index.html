Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito utilizando a tag <b> (não semântica; só forma! Jeito Errado).

Já nesta frase temos um termo em destaque usando a tag <strong> (semântica! Jeito Correto).

Itálico/Ênfase

Nesta frase temos um termo em itálico utilizando a tag <i> (não semântica; só forma! Jeito Errado).

Já nesta frase temos um termo em ênfase usando a tag <em> (semântica! Jeito Correto).

Marcador de Texto

Podemos criar, também, um texto com marca texto utilizando a tag <mark>.

E neste parágrafo temos um texto marcado no final.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno utilizando as tags <big> e <small> (Observar que a tag BIG está obsleta, enquanto que a small não).

Texto Deletado

Podemos marcar um texto como excluído/deletado para indicar que ele deve ser lido, mas desconsiderado, utilizando a tag <del>.

Texto Inserido/Sublinhado

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois utilizando a tag <ins>.

Existe, também, o sublinhado, com a tag <u> (não semântica)(cuidar: pode se tornar obsoleta).

Texto Sobrescrito

Para inserir coisas do tipo x20+3 (x elevado a 20 + 3), ou seja, com parte sobrescrita, basta utilizar a tag <sup>.

Texto Subscrito

Para inserir coisas do tipo H2O, ou seja, com parte subscrita, basta utilizar a tag <sub>.
