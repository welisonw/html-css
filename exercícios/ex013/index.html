Usando links externos

É possível utilizar links externos dentro do site utilizando a tag <a> de anchor (âncora). Dentro da tag, aparecerá o parâmetro href, que significa hipertext reference, que indica que é para onde o meu site vai:

Você pode acessar o meu repositório público no GitHub

Ainda, você pode acessar o meu canal no Youtube

Nos dois exemplos acima, ao clicar nos hiperlinks, os sites abrirão na mesma aba. Há a opção do hiperlink abrir em nova aba usando o parâmetro target (alvo) após a indicação do hiperlink, atribuindo o valor _blank (o valor _self abre o link na janela ou frame atual (padrão)):

Você pode acessar meu canal no Youtube.

Dentro da tag, ainda, é recomendado indicar se esse hiperlink é seu/interno ou de outra pessoa/externo (que vai pra fora do meu site). Para isso, basta utilizar o parâmetro rel (relative), atribuindo um dos seguintes valores:

  • next indica que o link é para a próxima parte do documento atual.
  • prev indica que o link é para a parte anterior do documento atual.
  • author indica que é um link para o site do autor do artigo atual.
  • external indica que é um link para outro site que não faz parte do site atual.
  • nofollow indica que é um link para um site não endossado, como um link pago.

Usando links internos

Links internos vão fazer relação com outra página que está dentro do nosso servidor e diz respeito à nossa própria página. Nesse caso, é necessária a criação de outras páginas:

Esta é a minha primeira página do site. Se você quiser, pode acessar também a minha segunda página

Quando eu faço um link interno, não preciso indicar a URL inteira, contanto que a "pag002" esteja dentro da mesma pasta do index.

Você pode também acessar a nossa página de notícias.

Links para download

Para adicionar arquivos para download, basta adicionar dentro da tag <a> o diretório do arquivo, seguido do parâmentro download, com o nome do arquivo e sua extensão, e do parâmentro type, indicando o tipo de arquivo está sendo baixado (lista das media types: Iana.org):
